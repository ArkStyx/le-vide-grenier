Une annonce à :

- 1.1 catégorie (immobilier, automobile, etc.)
- 0.n photo
- 1.1 description
- 1.1 date de publication
- 1.n critères
- 1.1 localisation (ville / code postal / departement
- 1.1 bouton "demander l'addresse précise"
- 1.1 contact
- 1.1 bouton "demande de contact"

Une annonce immobiliere a 1.n criteres :
- Type de bien
- Surface
- Surface du terrain
- Pièces
- Classe énergie
- GES
- Étage du bien
- Nombre de chambres
- Honoraires
- Référence

------------------------------------------------
------------------------------------------------

L'appli a :
- 0.n recherches
- 0.n Favoris
- 0.n Messages
- 1.1 connexion

------------------------------------------------
------------------------------------------------

un particulier/Une agence a :
- 0.n annonces
- 1.1 bouton "envoyer un message"
- 1.1 bouton "voir le numero"
- 1.1 statut (pro ou particulier)
